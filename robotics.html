Fast Robotics Project

Project Overview

Arduino-based control system for high-speed robotic navigation and obstacle avoidance.

Technical Details

  • Platform: Arduino Uno with custom shield
  • Sensors: Time of Flight , IMU, encoders
  • Control: PID controller with Kalman filtering
  • Programming: I2C, BLE, C++, Python

Key Achievements

  • Achieved stopping accuracy of <2cm
  • Implemented real-time obstacle avoidance
  • Orientation Control, with <5° precision

Skills Demonstrated

Embedded Systems • Control Theory • Sensor Fusion • C++ Programming • ROS • Real-time Systems

UR10e Advanced Path Planning

Project Overview

Advanced path planning and simulation for Universal Robots UR10e collaborative robot with ROS/MoveIt integration.

Technical Details

  • Robot: Universal Robots UR10e (10kg payload)
  • Software: ROS, MoveIt, Gazebo simulation
  • Planning: OMPL, RRT*, custom algorithms
  • Language: Python, C++, URScript

Key Achievements

  • Developed custom path planning algorithms for complex trajectories
  • Reduced cycle time by 35% through optimization
  • Created digital twin in Gazebo for simulation
  • Implemented real-time collision avoidance

Skills Demonstrated

ROS/MoveIt • Path Planning • Simulation (Gazebo) • URScript • Industrial Robotics • Motion Control • Python/C++
